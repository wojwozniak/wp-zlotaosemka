# wp-zlotaosemka

## Wstęp

- Nowa strona internetowa dla Złotej Ósemki. Lżejsza, szybsza, jaśniejsza i mniej zabugowana ^^.

## Technologie

- Wordpress, PHP, Javasript, CSS, HTML
